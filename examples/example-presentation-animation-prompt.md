# Example: Animated Presentation Prompt

```text
Use the uploaded Video Prompting kit.

I need an animated executive presentation about how our team should use AI for faster prototyping.

Audience:
Senior leaders and cross-functional stakeholders.

Goal:
Get approval for a pilot program.

Format:
PowerPoint, editable by the team.

Length:
8-10 slides, 7 minutes.

Brand:
Use our approved slide template, logo, colors, and typography.

Please:
1. Ask missing questions.
2. Create a slide-by-slide storyboard.
3. Recommend animation types for each slide.
4. Keep everything editable.
5. Put presenter intent in speaker notes.
```

## Good Motion Choices

- Morph for before/after workflow
- Appear for agenda and proof points
- Wipe for roadmap
- Fade for attention shifts
- No decorative animation that makes editing harder

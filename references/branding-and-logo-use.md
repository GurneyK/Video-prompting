# Branding And Logo Use

Brand assets should be used deliberately and traceably.

## Rules

- Use approved logos, not AI-recreated logos.
- Use existing brand colors and typography where available.
- Do not invent customer marks or partner logos.
- Track asset origins.
- Use a simplified logo treatment for small watermarks.
- Reserve big logo animation for intro or outro.

## When To Use A Logo

| Placement | Use |
|---|---|
| Intro | Establish who made it |
| Corner bug | Subtle persistent ownership |
| Outro | Final branded landing frame |
| Product UI | Only if it appears naturally in the product |

## Logo Animation Guidance

- Use vector SVG for draw-in effects.
- Use PNG only for static placement.
- Keep corner logos small and static.
- Do not animate the logo throughout the video.
- Avoid top-corner watermarks unless the layout requires it.

Review this video plan for brand safety. Check logo use, color consistency, typography, customer marks, and whether any asset origin is unclear.

Apply a company style reference to: $ARGUMENTS

1. Load the matching file from `templates/company-styles/` (see `templates/company-styles/README.md` for the full index if the exact company name isn't obvious, or if none of the 10 fit well and a closest match needs picking).
2. Read the file's IP disclaimer and hold to it strictly: pull pacing, restraint level, camera language, and narrative arc from the template; never copy an exact hex code, real logo, proprietary typeface, or copyrighted footage/copy — see `.claude/skills/brand-motion-system/references/reference-company-styles.md` for the full rule.
3. Cross-check the company's "best-fit audience" (in the template and the README index) against the actual audience lens for this project (`.claude/skills/creative-director/references/audience-lenses.md`, enterprise vs. startup). If they don't match, say so plainly and either recommend adapting harder than usual or picking a different reference — don't silently apply a mismatched register (e.g. Nike's kinetic energy onto a Unilever-style compliance-review deck).
4. Adapt, don't skin: swap in this project's own name, product, color choices in the same mood-family (not the literal reference palette), and copy. Apply the pacing/easing/hold-ratio/transition-grammar pattern from the template as the actual mechanical guidance.
5. Produce or update the style pack (`.claude/skills/brand-motion-system/references/style-pack-template.md` schema) with a "Reference Points" line stating which company's style was used as a starting point and what was explicitly changed to make it this project's own.
6. If the target is a storyboard rather than a style pack, apply the template's typical narrative arc and pacing directly to the storyboard's Motion/Visual/Audio columns, scene by scene.
7. Finish by running the Distinctiveness check: would someone familiar with the reference company recognize this as "inspired by" or as "a copy of" their work? If it reads as a copy, adjust before finishing.

# /company-style

Apply one of the top-10 company style-DNA reference templates as creative direction for a storyboard or brief.

Usage: `/company-style [company name, e.g. "anthropic", "apple", "spacex"] [target storyboard/brief]`

Apply a company style reference to: $ARGUMENTS

1. Load the matching file from `templates/company-styles/` (see `templates/company-styles/README.md` for the index).
2. Read the file's IP disclaimer and hold to it strictly: pull pacing, restraint level, camera language, and narrative arc; never copy an exact hex code, real logo, proprietary typeface, or copyrighted footage/copy.
3. Cross-check the company's best-fit audience against the actual project audience (enterprise vs. startup). If mismatched, say so and either adapt harder or pick a different reference.
4. Adapt, don't skin: swap in this project's own name, product, color choices in the same mood-family, and copy.
5. Produce or update the style pack with a "Reference Points" line stating what was borrowed and what was changed.
6. If the target is a storyboard, apply the template's narrative arc and pacing to the Motion/Visual/Audio columns directly.
7. Finish with the Distinctiveness check: would someone familiar with the reference company see this as "inspired by" or "a copy of" their work? Adjust if it reads as a copy.

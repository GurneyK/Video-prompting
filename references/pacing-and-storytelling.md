# Pacing And Storytelling

Pacing is not how fast things move. Pacing is how clearly the viewer can understand each idea before the next one arrives.

## The 60/40 Rule

For longer scenes, aim for roughly:

- 60 percent hold/read/settle time
- 40 percent motion/change time

Short punchy clips can be more motion-heavy, but business demos and presentations usually need more hold time.

## Signs A Video Is Rushed

- Text disappears before it can be read
- Multiple objects move at once
- Scene changes happen before the idea lands
- The viewer cannot summarize each scene
- The final frame appears mid-transition

## Fixes

- Add holds after text appears
- Remove scenes instead of speeding everything up
- Use fewer, stronger visuals
- Use one CTA
- Stagger related elements
- Make transitions slower and more intentional

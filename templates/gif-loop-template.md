```text
[topic]

[channel]

Constraints:
[duration, size, brand, file type, silent playback]

Before producing final prompts:
1. Recommend GIF or short MP4 loop.
2. Create a 3-5 beat loop plan.
3. Make sure the first and last frames match cleanly.
4. Keep text minimal and readable.
5. Provide export and QA notes.
```

## Loop Rules

- Best length: 2-6 seconds
- Start frame and end frame should match
- Avoid long text
- Avoid tiny UI details
- Optimize file size
- Prefer MP4 loop when quality matters

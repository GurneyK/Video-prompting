# Verification Checklist

Run this before sharing or publishing.

## Message

- [ ] The audience is clear.
- [ ] The goal is clear.
- [ ] The first 2-3 seconds make sense.
- [ ] The CTA is visible and specific.
- [ ] The piece works without narration.

## Brand And Assets

- [ ] Logo is approved.
- [ ] Colors and type match the brand.
- [ ] Screenshots are approved.
- [ ] No private/customer data is visible.
- [ ] Music, SFX, footage, and fonts are licensed.

## Visual QA

- [ ] Text is readable at final viewing size.
- [ ] No important object is cropped.
- [ ] Motion does not distract from the message.
- [ ] Every scene has a clear visual focus.
- [ ] Final frame lands cleanly.

## Platform

- [ ] Correct aspect ratio.
- [ ] Correct duration.
- [ ] Captions/silent version handled.
- [ ] File size is acceptable.
- [ ] Plays on the target device/platform.

## HTML Teaser Specific

- [ ] Opens in a real browser.
- [ ] Console has zero red errors.
- [ ] Scrubber works if present.
- [ ] Fullscreen works.
- [ ] Stage scales cleanly.
- [ ] Last frame holds.
- [ ] Hosted page works on desktop and mobile.

## AI Review Prompt

```text
Use this verification checklist to audit the storyboard/output. Report blockers, important fixes, and nice-to-have improvements separately.
```

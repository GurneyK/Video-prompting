# Example: HTML Video Teaser Prompt

```text
Use the uploaded Video Prompting kit, html-video-teaser-template, animations.jsx, scene-template.jsx, and build.mjs.

Build a 45-second HTML video teaser for a product called "Signal Desk".

Audience:
Internal product teams and leadership.

Goal:
Show how Signal Desk turns scattered inputs into a prioritized insight queue.

Scenes:
Intro, problem, ingest, prioritization, review workflow, outcome, outro.

Brand:
Use approved logo SVG. Use our product colors. Do not invent a new identity.

Output:
Mode B source and one generated teaser.html.

Before writing code:
1. Confirm Mode B is right.
2. Create a storyboard with durations and backgrounds.
3. Build only scene 1 first.
4. Verify in browser before adding more scenes.
5. Include fullscreen controls and final QA.
```

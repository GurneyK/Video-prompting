## Prompt

```text
Use the uploaded Video Prompting kit.

Create a showreel segment plan for:
[project/product]

Goal of this segment:
[goal]

Assets available:
[screens, clips, logos, mockups, finished links]

Context in reel:
[opening, middle proof, climax, outro]

Before making final prompts:
1. Ask missing questions.
2. Create a short storyboard with 3-6 shots.
3. Define transitions into and out of the segment.
4. Identify what should be shown as proof.
5. Give edit notes, sound notes, and QA risks.
```

## Segment Pattern

1. Establish the product/project
2. Show the most recognizable visual
3. Show one dynamic interaction or transformation
4. Land on proof/result
5. Exit with a transition that supports the next segment

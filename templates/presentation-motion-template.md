## Prompt

```text
Use the uploaded Video Prompting kit.

I need an animated presentation for:
[topic]

Audience:
[audience]

Goal:
[decision, understanding, approval, excitement]

Constraints:
[brand, length, editability, tool, required slides]

Before making final slides or motion instructions:
1. Ask missing questions.
2. Recommend whether this should be a deck, HTML teaser, MP4, or hybrid.
3. Create a slide-by-slide storyboard.
4. For each slide, define the message, layout, motion, speaker note, and QA concern.
5. Keep the animation editable and restrained.
```

## Default Motion Language

- Morph: before/after, UI state changes, object continuity
- Appear/Fade: reveal logic one beat at a time
- Wipe: timelines, process, left-to-right explanation
- Emphasis: only for the active object
- No excessive decorative motion

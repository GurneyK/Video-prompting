# Example: Product Demo Prompt

```text
Use the uploaded Video Prompting kit.

I need a 60-second product demo for a dashboard that monitors AI agent runs.

Audience:
Internal engineering, product, and leadership stakeholders.

Goal:
Show that the dashboard makes agent performance, evidence, and review status easy to understand.

Assets:
Screenshots of the dashboard, logo, sample metrics, no real customer data.

Final deliverable:
Hosted HTML teaser first, then possibly screen-recorded to MP4.

Before writing code:
1. Recommend HTML teaser Mode A or Mode B.
2. Create a storyboard with scene timing.
3. Identify which UI should be screenshot vs faux UI.
4. Build the first scene only after storyboard approval.
5. Include verification steps.
```

## Good AI Response Should Include

- Mode recommendation
- 5-7 scene storyboard
- Runtime estimate
- Asset list
- QA risks for fake data and confidentiality
